First a Little History

These two men to the left are named Mark Otto and Jacob Thorntonand they devlouped Bootstrap to be a easy webkit tool. They created bootstrap at twitter in 2010. During twitters first ever Hack week event Bootstraps exsploded in popularity! Everyone was shocked that web develoupers of all levels hopped on and started using bootstrap with no external guidance. The partners created something amazing and made code accessible for programers of all levels! Excited by their success they decided they were going to release Bootstrap to the public so coders across the world could jump in and start develouping. Since then Bootstraps has had around 20 diffrent releases. These releases include two major rewrites including V2 and V3. These rewrites have giving many new features and streamlined their tool. Now the two partners and their small team of 9 have all been working on Bootstraps together since day 1. These team continues to work full-time to streamline bootstraps, add more features and insure this webkit tool recivies all of the support it needs so it can continue to help all develoupers create the websites of their dreams.

Well worry not, you can rest peacefully knowing that Boot straps is Compatible with any browser you can think of! Safari, Chrome, Internet Explorer, Firefox, Opera and many more!
